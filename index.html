Path Finding using Multi Heuristic A* Algorithm


Instructions

This is a program to find the shortest path between two positions in a 2-dimensional grid with obstacles. To use it:
  • Enter start and goal locations.
  • Click on a grid element to block/unblock it.
  • Select the heuristic functions and their weights.
  • Press "Find Path" to see results.
